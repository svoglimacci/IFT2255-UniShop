Analyse

Les fonctionnalités pour les acheteurs, les revendeurs, les produits, les commandes, les évaluations, les métriques et les notifications doivent être informatisées. Les données sont stockées dans une base de données et des serveurs sont nécessaires pour gérer la base de données et l'authentification. Ceux-ci seront hébergés sur une plateforme en nuage comme AWS ou Firebase afin d'offrir un service sécuritaire aux utilisateurs et ce, peu importe leur localisation. UniShop interagit avec des systèmes externes de paiement en ligne et de suivi des commandes. Les utilisateurs interagissent via une interface de ligne de commande. Tout cela doit être accessible à partir de navigateurs web ou d'une application mobile dédiée.
